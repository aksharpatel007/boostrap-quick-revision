1. Grid System (BS5 6-Tier Responsive)

Bootstrap 5 introduces the `.col-xxl-*` breakpoint ($\ge 1400px$). Columns below demonstrate stacking on extra small screens and then breaking into $2$ or $4$ columns based on viewport size. **(UX: Increased Section Padding)**

Output Box
.col-md-6 .col-lg-3
.col-md-6 .col-lg-3
.col-md-6 .col-lg-3
.col-md-6 .col-lg-3
Code Box (UX: Larger Font/Padding)

    <!-- Columns stack on default (xs) -->
    <!-- 2 columns from Medium (768px) -->
    <!-- 4 columns from Large (992px) -->
    <div class="row g-2">
    <div class="col-12 col-md-6 col-lg-3">...</div>
    <div class="col-12 col-md-6 col-lg-3">...</div>
    <div class="col-12 col-md-6 col-lg-3">...</div>
    <div class="col-12 col-md-6 col-lg-3">...</div>
    </div>

3. Spacing Utilities (`m` & `p` for Margin & Padding)

Format: `{property}{sides}-{size}` or `{property}{sides}-{breakpoint}-{size}` (Chapter 6, Page 6)

Output Box

Parent (.p-1)

.p-5 .mt-3 .mb-5 .mx-auto (Padding 5, Margin Top 3, Margin Bottom 5, Center)
.py-4 (Padding Top/Bottom 4)
Code Box

    <!-- Padding all sides (p-5) -->
    <div class="p-5">...</div>

    <!-- Margin top (mt-3) and bottom (mb-5) -->
    <div class="mt-3 mb-5">...</div>

    <!-- Padding on Y-axis (py-4) -->
    <div class="py-4">...</div>

    <!-- Center horizontally (mx-auto) -->
    <div class="mx-auto">...</div>
